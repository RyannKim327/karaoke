# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0-test] - 2026-05-20

### Added
- Initial release of the tested version.
- Integrated **Pitch-based Scoring System** using YIN algorithm for real-time singing feedback.
- **QR Code integration** for easy mobile-to-room connection.
- Project logo added to the documentation and UI.
- Detailed usage instructions and cultural context in README.

### Changed
- **Major Update:** Switched search engine and music source from YouTube to **Tubidy** for better performance and reliability.
- Refined UI with a modern, centered header in documentation.
- Updated project description to highlight its roots as a digitalized KTV machine.
- Synchronized versions across all packages to `1.2.0-test`.
- Improved room management and synchronization logic.
